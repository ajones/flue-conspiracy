# Vault — Agent Instructions

This is a shared knowledge vault. When working here:

- New material goes into `inbox/` unless a matching category folder already exists.
- Use the `NN-name` convention when creating new category folders.
- Filenames should be lowercase, hyphen-separated, and descriptive.
- Prefer Markdown files. Keep one topic per file.
- When sorting inbox items, move the file — do not copy.
