# Vault

A general-purpose knowledge base. Drop things in `inbox/` and sort them into categories over time.

## Structure

```
vault/
  inbox/    — unsorted incoming notes, links, and scraps
  NN-name/  — category folders added as content grows
```

## Conventions

- Folders follow the `NN-name` pattern so they sort consistently.
- New material lands in `inbox/` first; sort into categories when ready.
- Filenames should be descriptive and lowercase with hyphens.
